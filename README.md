GIT
===
* git clone https://github.com/aptitud/SimpleDashboard.git

Build and run
=============
* npm install
* npm start

Heroku
======

SimpleDashboard is hosted on Heroku. To deploy from github, do:

* git clone https://github.com/aptitud/SimpleDashboard.git
* cd SimpleDashboard
* heroku git:remote -a safe-brook-5766
* git push heroku master
